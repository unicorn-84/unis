# Heading

Heading text.

## Usage

```javascript
import { Heading } from '@web-uni/unis';
<Heading>Heading text</Heading>
```

## Properties

### level

The heading level.

*Type:* "h1" | "h2" | "h3" | "h4" | "h5" | "h6"

*Required:* false

*Default:* "h1"

### variant

A heading text can define its look via defined variants.

*Type:* "title" | "h2" | "h3" | "h4"

*Required:* false

*Default:* "title"

## Intrinsic element

```
h1 | h2 | h3 | h4 | h5 | h6
```

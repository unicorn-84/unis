# Text

Arbitrary text.

## Usage

```javascript
import { Text } from '@web-uni/unis';
<Text>Arbitrary text</Text>
```

## Properties

### variant

A arbitrary text can define its look via defined variants.

*Type:* "text" | "strong" | "small" | "button"

*Required:* false

*Default:* "text"

### tag

The DOM tag.

*Type:* "span"

*Required:* false

*Default:* "span"

span

# Link

A text link.

## Usage

```javascript
import { Link } from '@web-uni/unis';
<Link href="#">Text</Link>
```

## Properties

### disabled

Whether the anchor is disabled.

*Type:* "true" | "false"

*Required:* false

*Default:* false

### underline

Controls when the link should have an underline.

*Type:* "always" | "none"

*Required:* false

*Default:*

## Intrinsic element

```
a
```

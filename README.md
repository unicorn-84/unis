# Unis

UI components for react.js websites and apps

## Documentation

Visit the Unis website for more information.

## Install

You can install Unis using either of the methods given below.

For npm users:

```shell
  npm i unicorn-84/unis
```

For Yarn users:

```shell
  yarn add unicorn-84/unis
```
